Sockets & compatibility

  • Red aug → Red, Purple, Orange sockets
  • Blue aug → Blue, Purple, Green sockets
  • Yellow aug → Yellow, Orange, Green sockets
  • Purple/Orange/Green aug → same-color only
  • Colorless aug → any colored socket; Colorless socket accepts only Colorless
  • Sun/Moon aug → Sun/Moon sockets only
  • Named lines (Lamordia/IoD) → exact match of socket type

“Sheltering” contributes to Physical Sheltering (PRR) and Magical Sheltering (MRR) with the same modifier; each competes with other entries of that stat + modifier. Mythic stacks across sources.
